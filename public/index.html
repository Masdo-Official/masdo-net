❓ FAQ

Pertanyaan Umum

Jawaban untuk pertanyaan yang sering ditanyakan seputar sistem absensi

Bagaimana cara login ke sistem?

▼

Siswa dapat login menggunakan kode siswa yang telah diberikan (format: XI9-XXX) dan password yang telah didistribusikan oleh admin kelas. Jika lupa password, hubungi admin kelas untuk reset.

Apakah data absensi disimpan dengan aman?

▼

Ya, semua data disimpan dengan aman menggunakan Google Sheets dengan autentikasi service account. Password siswa di-hash menggunakan bcrypt untuk keamanan maksimal. Data tidak akan dibagikan kepada pihak ketiga tanpa izin.

Bagaimana cara admin menginput absensi?

▼

Admin dapat login ke panel admin, pilih menu "Input Absensi", tentukan tanggal, lalu muat daftar siswa. Setelah itu tandai status kehadiran setiap siswa (Hadir/Izin/Sakit/Alpa) dan tambahkan keterangan jika diperlukan. Klik "Simpan Absensi" untuk menyimpan ke database.

Apakah bisa export data absensi?

▼

Ya! Admin dapat mengexport semua data absensi ke format CSV melalui menu "Export Data" di panel admin. File CSV dapat dibuka di Excel atau Google Sheets untuk analisis lebih lanjut.

Apa yang harus dilakukan jika sistem error?

▼

Coba refresh halaman terlebih dahulu. Jika masih error, periksa koneksi internet Anda. Untuk masalah teknis yang persisten, hubungi admin kelas atau pemilik sistem untuk bantuan troubleshooting.

Apakah sistem ini bisa diakses dari HP?

▼

Tentu saja! Sistem ini menggunakan responsive design yang optimal untuk berbagai ukuran layar, termasuk smartphone, tablet, dan desktop. Anda bisa mengakses dari browser HP kapan saja dan dimana saja.
